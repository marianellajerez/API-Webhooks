📖 Guía paso a paso
  1. 1. Asegúrate que el servidor esté corriendo: npm start
  2. 2. Elige un ID de documento: (ej: mi-prueba-001)
  3. 3. Suscríbete en esta página: Escribe el ID y presiona "Suscribirse" o Enter
    • Verás tarjeta VERDE: SUSCRIPCION
  4. 4. En PowerShell (espera 2 segundos): .\scripts\test-webhook.ps1 mi-prueba-001
    • Verás tarjeta AZUL: document:statusChanged
  5. 5. Para probar incidencia (firma inválida): .\scripts\test-incident.ps1 mi-prueba-001
    • Verás tarjeta ROJA: integration:incident
  6. ⚠️ Importante: Primero suscríbete en esta página, luego ejecuta los scripts en PowerShell.
  7. Limpiar: Presiona "Limpiar eventos"
